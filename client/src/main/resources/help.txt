О программме.
Облачное хранилище, позволяющее хранить и отправлять на сервер и получать от сервера файлы.

-> В левом окне располагается список файлов пользователя на компьютере.
-> В правом окне располагается список файлов, хранящихся на сервере.

Кнопки действий над файлами (сверху вниз):
-> Отправить файл на сервер
-> Скачать файл с сервера в локальную директорию пользователя
-> Обновление списка файлов
-> Удаление выбранного файла
-> Создание новой директории в локальной директории пользователя
-> Создание новой директории на сервере
-> Для удаления или перемещения файлов выделите нужный файл и нажмите правую кнопку мыши. В
появившемся окне выберите нужную команду.
